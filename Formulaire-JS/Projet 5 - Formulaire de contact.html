Projet 5 - Formulaire de contact

Introduction

Nous continuons nos réaliastion avec notre cinquième projet qui sera un formulaire de contact complet.

Dans ce cinquième cas nous allons donc créer un formulaire de contact assez complet avec des champs pour le nom, le prénom, un email, une selct box et un champ textaréa pour y laisser des commentaires.

La page final ressemblera à ceci :

Pour la réaliseation de cette page nous aurons besoin d’utiliser 3 langages HTML/CSS et JavaScript.

Structure HTML

Avant de commencer vous pouvez télécharger les documents source de ce formulaire en cliquant sur ce lien et dans lequel vous trouverez les fichiers de base (vide bien sûr) et le logo à utiliser dans le dossier 1.5 ContactezNous.

                    <input type="text" name="nom" id="nom" required maxlength="35" class="flexInp" placeholder="Entrez votre nom">

    </div>

</body>
</html>

Explication de la page

Cette page HTML représente notre formulaire de contact qui permet aux utilisateurs de nous contacter sur via notre site web.

La structure du formulaire est divisée en plusieurs éléments HTML, chaque élément ayant un objectif précis :

La première division est la balise div avec la classe boxForm. Cette balise contient tout le formulaire et est utilisée pour ajouter un style spécifique à l’ensemble du formulaire.

La balise h1 contient le titre “Contactez-nous” et le logo de l’entreprise sous la balise span avec la classe logoTitre.

La balise form est utilisée pour encadrer le formulaire et permettre le traitement des données entrées.

Le premier champ de saisie est une balise input de type email avec le nom email, l’id email, la classe emailInp, un attribut required et un placeholder Entrez votre email. La balise label associée à cet élément a une classe labelEmail et est utilisée pour fournir une étiquette pour ce champ de saisie.

La division suivante est utilisée pour regrouper deux champs de saisie dans une même ligne, avec une division div qui a une classe flex2Inputs.

Dans la division flex2Inputs, il y a deux balises input de type text avec les noms prenom et nom, les ids prenom et nom, les classes flexInp, un attribut required, un attribut maxlength et des placeholders pour entrer le prénom et le nom.

La balise label associée à chaque champ de saisie a une classe labelFlex et est utilisée pour fournir une étiquette pour ces champs de saisie.

La balise select est utilisée pour créer une liste déroulante avec un nom select, un id inpSelect et des options pour sélectionner une catégorie. La première option est utilisée pour indiquer que la sélection doit être faite.

La balise label associée à cet élément a une classe labelSelect et est utilisée pour fournir une étiquette pour cette liste déroulante.

La balise textarea est utilisée pour créer une zone de texte pour saisir un message avec un nom txt, un id txtArea, un nombre de colonnes et de lignes.

Enfin la balise button est utilisée pour envoyer le formulaire en cliquant sur le bouton “Envoyer”.

Réalisation du style

N’ésiter pas à aller voir le résultat de votre page après chaque insertion de style css.

Si vous observer votre page, ce n’est pas encore ça ! C’est pourquoi nous allons maintenant voir comment styliser tout ça avec du CSS.

Donc c’est partie comme d’habitude on ouvre notre feuille de style “style.css”.

Comme toujours on commence par le style de base de notre page, les sélecteurs universel :






*, ::before, ::after {
    box-sizing: border-box;
    margin: 0;
    padding: 0;
}

L’ensemble des éléments de la page (*) auront la propriété box-sizing avec la valeur border-box, ce qui signifie que la taille totale de l’élément inclut sa bordure et son remplissage.

L’ensemble des éléments de la page (*) auront une marge et un remplissage de 0.

Style du sélecteur body








body {
    background: #f1f1f1;
    display: flex;
    justify-content: center;
    height: 100vh;
    font-family: Arial, Helvetica, sans-serif;
}

background: #f1f1f1; : Définit la couleur de fond de la page à gris clair.
display: flex; : Définit la boîte de la page en tant que conteneur flexible.
justify-content: center; : Centre le contenu horizontalement.
height: 100vh; : Définit la hauteur de la page à 100% de la hauteur de la vue.

Style du sélecteur .boxForm









.boxForm {
    margin-top: 30px;
    padding: 50px;
    width: 700px;
    height: 740px;
    background: #333;
    border-radius: 5px;
}

margin-top: 30px; : Ajoute une marge en haut de 30px à la boîte de formulaire.
padding: 50px; : Ajoute un remplissage de 50px autour du contenu du formulaire.
width: 700px; : Définit la largeur de la boîte de formulaire à 700px.
height: 740px; : Définit la hauteur de la boîte de formulaire à 740px.
background: #333; : Définit la couleur de fond de la boîte de formulaire à noir foncé.
border-radius: 5px; : Arrondit les coins de la boîte de formulaire.

Style du sélecteur h1







h1 {
    margin-top: 0;
    font-size: 50px;
    text-align: center;

margin-top: 0; : Supprime la marge en haut de l’en-tête.
font-size: 50px; : Définit la taille de police de l’en-tête à 50px.
text-align: center; : Centre le texte horizontalement.
color: #f1f1f1; : Définit la couleur du texte de l’en-tête à blanc.

 h1 span img {

    position: relative;
    top: 10px;

width: 50px; : définit la largeur de l’image dans le titre à 50px.
position: relative; : positionne l’image relativement à son conteneur parent.
top: 10px; : déplace l’image de 10px vers le bas par rapport à sa position par défaut.

border: 0; : Supprime les bordures des éléments.

background: lightgreen; : Définit la couleur de fond du bouton en vert clair lorsqu’il est survolé par la souris.
