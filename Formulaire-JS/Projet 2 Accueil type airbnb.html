Projet 2 Accueil type airbnb

Introduction

Nous continuons nos réaliastion avec notre second projet qui est une page d’accueil style airbnb.

Dans ce second cas nous allons donc créer une page d’accueil, avec un champ (input) permettant simplement de rechercher un lieu.

Le formulaire final ressemblera à ceci :

Structure HTML

Avant de commencer vous pouvez télécharger les documents source de ce formulaire en cliquant sur ce lien et dans lequel vous trouverez les fichiers de base (vide bien sûr) et le logo à utiliser dans le dossier 1.2 AccueilStyleAirbnb.

Maintenant que nous avons tout ce qu’il nous faut pour commencer notre formulaire nous allons attaquer le HTML.




































<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Accueil Recherche</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
  
    <div class="section">

        <div class="logoHautGauche">Lorem, ipsum dolor.</div>

        <div class="listeTopDroite">
            <div class="item">Lorem</div>
            <div class="item">Lorem</div>
            <div class="item">Lorem</div>
        </div>

        <h1>Découvre le monde.</h1>

        <form class="flexForm">

            <label for="search">
                <img src="icone.svg" alt="icone">
            </label>
            <input type="search" name="recherche" id="search" placeholder="Où veux-tu aller?">
            <button>Valider</button>
        </form>

    </div>

</body>
</html>

Explications

La première div a pour classe “section”. Elle contient plusieurs autres éléments.

La deuxième div a pour classe “logoHautGauche”. C’est un texte qui apparaît en haut à gauche de la section.

La troisième div a pour classe “listeTopDroite”. Elle contient trois éléments de classe “item” qui forment une liste à droite en haut de la section.

Le h1 contient le texte “Découvre le monde.” qui est affiché sous la première div.

Le form a pour classe “flexForm”. Il contient une étiquette (label) et un champ de recherche (input) qui permettent de rechercher une destination de voyage. L’input a un identifiant “search” et un placeholder “Où veux-tu aller?”. Le bouton “Valider” permet de lancer la recherche.

Réalisation du style

Si vous observer votre page, elle ne ressemble pas encore à grand chose ! C’est pourquoi nous allons maintenant voir comment styliser tout ça avec du CSS.

Donc c’est partie on ouvre notre feuille de style “style.css”.

On commence par le style de base de notre page donc le body :








body {
    margin: 0;
    padding: 0;
    background: #333;
    height: 100vh;
    font-family: Arial, Helvetica, sans-serif;
}

Explications

Le code CSS fourni contient les propriétés suivantes pour le sélecteur body :

margin: 0;: Supprime la marge par défaut du corps de la page, qui est ajoutée par le navigateur.

padding: 0;: Supprime le remplissage par défaut du corps de la page, qui est ajouté par le navigateur.

background: #333;: Définit la couleur de fond du corps de la page en utilisant une valeur de couleur hexadécimale #333, qui correspond à une nuance de gris foncé.

height: 100vh;: Définit la hauteur du corps de la page en utilisant une valeur de hauteur de 100vh, ce qui signifie que le corps de la page occupera la totalité de la hauteur de la fenêtre d’affichage.

font-family: Arial, Helvetica, sans-serif;: Définit la police de caractères par défaut pour tout le contenu de la page en utilisant la police de caractères Arial, Helvetica et sans-serif en dernier recours.

On continue donc avec la section d’accueil :













.section {
    width: 100%;
    height: 100%;
    display: flex;
    justify-content: center;
    align-items: center;
    flex-direction: column;
    background: url('bgsombre.jpg');
    background-repeat: no-repeat;
    background-size: cover;
    background-position: center;
}

Le style CSS ci-dessus s’applique à l’élément HTML avec la classe .section. Il comprend les propriétés suivantes :

width: 100%; : définit la largeur de l’élément .section à 100% de la largeur de son conteneur.

height: 100%; : définit la hauteur de l’élément .section à 100% de la hauteur de son conteneur.

display: flex; : définit le mode d’affichage de l’élément .section en tant que conteneur flexible.

justify-content: center; : centre l’alignement horizontal de tous les éléments enfants de .section.

align-items: center; : centre l’alignement vertical de tous les éléments enfants de .section.

flex-direction: column; : définit l’axe principal du conteneur flexible comme étant vertical.

background: url('bgsombre.jpg'); : définit l’image de fond pour l’élément .section.

background-repeat: no-repeat; : empêche la répétition de l’image de fond.

background-size: cover; : redimensionne l’image de fond pour couvrir l’ensemble de l’élément .section.

background-position: center; : centre l’image de fond dans l’élément .section.

Ensemble, ces propriétés créent un conteneur flexible centré horizontalement et verticalement, avec notre image de fond redimensionnée pour couvrir l’ensemble de l’élément. Le texte et les autres éléments enfants de .section seront positionnés au centre du conteneur.

Nous allons maintenant styliser la classe logoHautGauche.








.logoHautGauche {
    position: absolute;
    top: 10px;
    left: 10px;
    font-size: 22px;
    color: #f1f1f1;
}

N’ésiter pas à aller voir le résultat de votre page après chaque insertion de style css.

Explications

Le sélecteur .logoHautGauche cible l’élément HTML qui a la classe “logoHautGauche”. Les propriétés CSS appliquées à cet élément sont les suivantes :

position: absolute; : place l’élément en position absolue par rapport à son conteneur. Cela signifie que la position de l’élément est fixée, indépendamment de la position des autres éléments sur la page.

top: 10px; : place l’élément à 10 pixels du haut de son conteneur.
left: 10px; : place l’élément à 10 pixels de la gauche de son conteneur.
font-size: 22px; : définit la taille de la police de caractères à 22 pixels.
color: #f1f1f1; : définit la couleur du texte à un blanc cassé.

En combinant ces propriétés, l’élément ayant la classe “logoHautGauche” est positionné en haut à gauche de son conteneur avec une taille de police de 22 pixels et une couleur de texte blanche cassé.

On passe maintenant à la classe listeTopdroite de notre page.














.listeTopDroite {
    display: flex;
    position: absolute;
    top: 10px;
    right: 50px;
}

.item {
    margin: 10px;
    color: #f1f1f1;
    cursor: pointer;
}

N’ésiter pas à aller voir le résultat de votre page après chaque insertion de style css.

Explications

Le code CSS ci-dessus définit les styles pour la liste d’éléments alignés horizontalement dans le coin supérieur droit. Les propriétés utilisées sont les suivantes:

display: flex;: Cela définit le conteneur de la liste comme un conteneur flexible. Cela signifie que les éléments enfants peuvent être facilement alignés horizontalement.

position: absolute;: Cela définit la position de la liste comme étant absolue par rapport à la section parente.

top: 10px;: Cela définit la distance en pixels entre le haut de la liste et le haut de la section parente.

right: 50px;: Cela définit la distance en pixels entre la droite de la
liste et la droite de la section parente.

.item définit les styles pour chaque élément de la liste. Ils ont une marge de 10 pixels, une couleur de texte blanche #f1f1f1 et un curseur personnalisé lorsque l’utilisateur passe la souris dessus avec cursor: pointer.

En utilisant ces propriétés, la liste sera positionnée en haut à droite de la section parente et sera alignée horizontalement.

Nous passons donc à notre titre







h1 {
    font-size: 110px;
    color: #fff;
    margin-bottom: 10px;
    text-align: center;
}

N’ésiter pas à aller voir le résultat de votre page après chaque insertion de style css.

Explications

Le code CSS ci-dessus applique des styles à notre titre h1 présent dans le code HTML. Voici ce que font les différentes propriétés CSS :

font-size: 110px;: Définit la taille de la police de caractères utilisée pour le titre, ici définie à 110 pixels.

color: #fff;: Définit la couleur de la police de caractères, ici définie sur blanc #fff.

margin-bottom: 10px;: Définit la marge en bas du titre, ici définie à 10 pixels.

text-align: center;: définit l’alignement horizontal du texte dans le titre, ici défini au centre.

Passons maintenant à la partie formulaire de notre page :

.flexForm est le sélecteur pour le formulaire et utilise display: flex pour permettre à ses éléments enfants de se positionner de manière flexible.

z-index: 10 et position: relative sont utilisés pour positionner le formulaire au-dessus de l’image de fond de la section. width: 500px définit la largeur du formulaire.

box-shadow ajoute une ombre au formulaire pour le faire ressortir légèrement.

N’ésiter pas à aller voir le résultat de votre page après chaque insertion de style css.

.flexForm > * est le sélecteur pour tous les éléments enfants directs du formulaire (étiquettes, entrées de texte, boutons), et définit certains styles partagés par tous ces éléments.

Ils ont border: 0 pour supprimer toute bordure par défaut, padding: 0 0 0 10px pour ajouter un peu d’espace sur le côté gauche, background: #fff pour donner un arrière-plan blanc, line-height: 50px pour centrer le texte vertical, font-size: 16px pour définir la taille de police, border-radius: 0 pour supprimer tout arrondi de bordure, et outline: 0 pour supprimer la bordure en pointillé qui apparaît lorsque l’on clique sur un champ.
