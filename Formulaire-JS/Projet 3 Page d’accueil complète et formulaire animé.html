Explications

La page HTML ci-dessus est composée d’un conteneur global contenant deux panneaux, un à gauche et un à droite. Le panneau de gauche contient un titre, un formulaire de connexion, et un lien pour récupérer un mot de passe perdu.

Le panneau de droite contient une illustration et un sous-titre.

Le conteneur global est défini avec la classe CSS “contGlobal”. Le panneau de gauche est défini avec la classe CSS “panneauGauche”. Le titre “Techno Stock” est défini avec la balise HTML h1 et est écrit en gras avec l’élément “strong”. Le formulaire de connexion est défini avec la classe CSS “formBloc” et contient trois éléments de formulaire.

Chaque élément de formulaire est défini avec la classe CSS “formGroupe” et contient une étiquette avec la balise HTML label et un champ de saisie avec la balise HTML input. Le champ de saisie pour le mot de passe est masqué. Le bouton de soumission du formulaire est défini avec la classe CSS “buttonSub”.

Le panneau de droite est défini avec la classe CSS “panneauDroit”. Il contient une illustration définie avec la classe CSS “illustration” et une image avec la balise HTML img. Le sous-titre est défini avec la balise HTML h2.

Réalisation du style

N’ésiter pas à aller voir le résultat de votre page après chaque insertion de style css.

Si vous observer votre page, ce n’est pas encore ça ! C’est pourquoi nous allons maintenant voir comment styliser tout ça avec du CSS.

Donc c’est partie on ouvre notre feuille de style “style.css”.

On commence par le style de base de notre page, les sélecteurs universel et le body :










*, ::before, ::after {
    box-sizing: border-box;
    margin: 0;
    padding: 0;
}

body {
    font-family: montserrat, sans-serif;
}

Explications

L’ensemble des éléments de la page (*) auront la propriété box-sizing avec la valeur border-box, ce qui signifie que la taille totale de l’élément inclut sa bordure et son remplissage.

L’ensemble des éléments de la page (*) auront une marge et un remplissage de 0.
Sélecteur de balise :

L’élément body aura la police de caractères Montserrat ou une police sans-serif.

Les éléments avec la classe panneauGauche auront une flexibilité de 1 pour leur facteur de flexibilité de base, 1 pour leur facteur de flexibilité maximale et une largeur de 33,333% de leur conteneur.

Petit apparteil
L’utilisation de flex ici est un raccourcit de la propriété flex comme pour les propriété background, padding, etc… Ici nous utilisons la propriété flex avec trois propriétés : flex: 1 1 33.333%; cela aurait été équivalant de ceci :




flex-grow: 1;
flex-shrink: 1;
flex-basis: 33.333%;

Quelques explications

flex-grow: 1; définit la capacité de l’élément à se développer lorsqu’il y a de l’espace disponible dans le conteneur parent. Dans ce cas, l’élément peut occuper tout l’espace disponible.

flex-shrink: 1; définit la capacité de l’élément à se rétrécir lorsqu’il y a un manque d’espace dans le conteneur parent. Dans ce cas, l’élément peut se rétrécir pour s’adapter à l’espace disponible.

flex-basis: 33.333%; définit la taille de base de l’élément avant que l’espace disponible soit réparti entre les éléments flexibles. Dans ce cas, la taille de base est définie à 33.333% de la largeur du conteneur parent.
Ils auront également un remplissage de 25px.

Explications

Donc flex: 1 1 33.333%; définit la propriété flexbox pour cet élément. Cela signifie que l’élément aura une flexibilité de 1, ce qui signifie qu’il pourra se rétrécir ou se développer en fonction de l’espace disponible dans le conteneur parent. Les valeurs 1 et 33.333% indiquent que l’élément aura une taille flexible, mais qu’il doit occuper initialement un tiers de la largeur du conteneur parent.

padding: 25px; Ajoute un remplissage de 25 pixels à tous les côtés de l’élément, ce qui créera un espace entre le contenu et la bordure de l’élément.

Animation des labels descendants directs des éléments de “formGroupe”

Pour gérer l’animation sur nos input nous allons utiliser pour cela le Javascript et le CSS.

Commençons par le Javascript. Nous allons donc ajouter une balise script à la fin de notre page HTML comme on a l’habitude de le faire.
















<script>
   const input_fields = document.querySelectorAll('input');

   for (let i = 0; i < input_fields.length; i++) {
        let field = input_fields[i];
        field.addEventListener('input', function(e){

          if(e.target.value != ""){
            e.target.parentNode.classList.add('animation');
          } else if (e.target.value == "") {
            e.target.parentNode.classList.remove('animation');
          }
        })
     }
</script>

Le script Javascript ci-dessus effectue les actions suivantes :

On sélectionne tous les éléments HTML de type input en utilisant la méthode querySelectorAll() et stocke ces éléments dans la variable input_fields.

Il boucle à travers chaque élément dans input_fields en utilisant une boucle for et stocke chaque élément dans la variable field.

Pour chaque élément field, il ajoute un écouteur d’événements input qui écoute les modifications du contenu de l’élément.

Lorsque le contenu de l’élément change, la fonction callback associée à l’écouteur d’événements est appelée.

La fonction vérifie si la valeur de l’élément est différente d’une chaîne vide (“”). Si c’est le cas, elle ajoute la classe “animation” à l’élément parent de l’élément en cours de modification (l’élément “field”). Sinon, elle supprime la classe “animation” de l’élément parent.

En résumé, le script ajoute une classe “animation” à l’élément parent de chaque champ de saisie (input) dès que l’utilisateur commence à taper dans le champ. Cette classe peut ensuite être utilisée pour animer l’étiquette (label) de l’élément ou pour ajouter d’autres effets visuels.
